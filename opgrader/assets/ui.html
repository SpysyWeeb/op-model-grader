op-model-grader

Grade your openpilot model against your own manual driving. Local-only server — your token never leaves this machine except to api/athena.comma.ai.

Connect your comma account

This tool needs a comma API token (JWT) to list your routes and download rlogs. Get one:

  • Open jwt.comma.ai, sign in, and copy the token, or
  • if you've used comma's CLI tooling before, the token is already in ~/.comma/auth.json.
